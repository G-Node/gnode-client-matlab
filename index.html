gnode-client-matlab is a MATLABx toolbox that connects your workbench to the G-Node datastore for electrophysiological recordings. It is MIT licensed, and open source.

Latest Version: 0.1-alpha

Getting started

% Put the toolbox on your path. Then do:
      
>>> c = gnode.init('/my/configuration/file.json');
>>> gnode.authenticate(c);
      [GNODE] SUCCESS

% Back to work
>>> plot(gnode.getdata(c, 'analogsignal_214'));

Core functions

>>> c = gnode.init(CONFIGURATION_FILE);
>>> gnode.authenticate(c);
>>> gnode.set_mode(c, 'command_line');

>>> gnode.list(c, 'type')
>>> gnode.children(c, 'id')
>>> gnode.parent(c, 'id')

>>> gnode.get_object(c, 'id')
>>> gnode.get_info(c, 'id')
>>> gnode.get_data(c, 'id')
>>> gnode.get_recursive(c, 'id')

>>>  signal = gnode.get_object(c, 'analogsignal_324');
>>>  signal.data = transform(signal.data);
>>>  gnode.put(c, signal)

>>>  gnode.start_transaction(c)
>>> % (...)
>>>  gnode.do(c)

>>>  gnode.kill(c)
