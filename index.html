gnode-client-matlab is a MATLAB toolbox that connects your workbench to the G-Node datastore for electrophysiological recordings. It is MIT licensed, and open source.

Latest Version: 0.1-alpha

Getting started

% Put the toolbox on your path. Then do:
      
>>> c = gnode.init('/my/configuration/file.json');
>>> gnode.authenticate(c);
      [GNODE] SUCCESS

% Back to work
>>> plot(gnode.getdata(c, 'analogsignal_214'));

Core functions

% Basics
>>> c = gnode.init('/path/to/configuration.json');
>>>gnode.authenticate(c);

% Modify session settings (e.g., logging/cache behaviour)
>>> gnode.set_mode(c, 'command_line');
>>> gnode.set_cache(c, 'mongodb@localhost:8888', 'all')

% Explore your available data
>>> gnode.list(c, 'type')
>>> gnode.children(c, 'id')
>>> gnode.parent(c, 'id')

% Pull NEO objects into local structures
>>> object = gnode.get_object(c, 'id')
>>> plot(object.signal.data)
>>> object.signal.units
ans =

      mV

>>> gnode.get_info(c, 'id')
>>> gnode.get_data(c, 'id')

% Fully *local* data representation
>>> gnode.get_recursive(c, 'id')

% Get, manipulate and stash your results at G-Node
>>> signal = gnode.get_object(c, 'analogsignal_324');
>>> signal.data = transform(signal.data);
>>> gnode.put(c, signal)

% Streamline workflow with transactions
>>> gnode.start_transaction(c)
% (...)
>>> gnode.do(c)

% Clean up
>>> gnode.kill(c)
